## 设计一个页面能显示当前天气
[FreeCodeCamp课程链接](http://freecodecamp.cn/challenges/show-the-local-weather)


###功能
设计一个网页，可以通过ajax访问一个api, 并显示local weather。 bonus: 实现'F和'C的转换

###步骤
按照结构->样式->功能 进行开发，重点在功能。

### 结构
简单结构，背景图片，标题，当前温度；地点，天气（晴阴雨）；按钮（转换'F和'C）
